**IPA Internship Management System** is a full-stack web application that helps manage internships in schools or institutions.  
It allows:

- **Students** to track tasks, update profiles, and receive ratings.  
- **Supervisors** to assign tasks, evaluate students, and provide feedback.  
- **Teachers** to monitor progress and supervise the program.  

The system also sends **automated profile completion emails** and supports secure authentication.

---

## Features

- Role-based authentication (Student, Supervisor, Teacher, Admin)  
- Task management and progress tracking  
- Ratings and feedback system  
- Messaging between students and supervisors  
- Automated email notifications for profile completion  
- PostgreSQL database managed with Prisma ORM  

---

## Tech Stack

- **Frontend:** Next.js 16, Tailwind CSS  
- **Backend:** Next.js API Routes, Node.js  
- **Database:** PostgreSQL, Prisma ORM  
- **Email:** Nodemailer
